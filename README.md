## 🔑 주요기능

- Public
  - Blog: 블로그 포스트 뷰 및 작성 기능
  - Portfolio: 프로젝트 리스트 및 상세 정보 제공
- Private
  - TodoList
  - AccountBook

```
├── api                 # API 라우트
├── assets
├── components
├── data
├── lib
├── models
├── server
├── types
└── utils
```
